# Progetto-Sito-Gamestore
Questo sito web è un Game Store gestito tramite form HTML e PHP. L'utente sceglie una categoria di videogiochi (es. FPS, Sport) nella index.html. La richiesta passa a gamestore.php che, usando cicli e switch, estrae i dati dagli array e stampa a schermo una tabella con titoli, PEGI e immagini, il tutto stilizzato in stile cyberpunk con style.css.
